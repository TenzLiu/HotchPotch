# HotchPotch
一款个人项目  

主要模块有首页、新闻、视频、图片，除此之外还有指纹识别登录、分享、推送、二维码、热修复...等等。目前这些功能已完成，但后续还会继续添加新功能进来。  

项目中所用到的第三方库都已经在gradle里面有链接地址。

下载apk体验：https://fir.im/c3dh
